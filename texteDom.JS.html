Poderosa ferramenta que está dentro do navegador, um conjunto de objetos do JavaScript no tratamento de componentes visuais

Além disso também aprenderemos como funciona uma Árvore DOM

Modelo de Objetos para documentos, basicamente é um conjunto objetos dentro do seu navegador que vai dar acesso aos componentes internos do web site, ele está presente quando esta rodando JavaScript dentro do navegador.

A árvore DOM começa da raiz e essa raiz dentro do navegador é chamada de WINDOW, TUDO DENTRO DO JAVASCRIPT ESTÁ DENTRO DE UM OBJETO CHAMADO WINDOW (JANELA).
